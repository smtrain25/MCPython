# CST_01 — Click to Clear

Non-agentic click-elimination puzzle.

## How to Play

Coloured squares are scattered on the grid. Click each one to remove it.
Clear all squares to advance to the next level.

## Controls

| Key | Action |
|-----|--------|
| Click | Remove target square |

## Levels

5 levels with increasing grid size (8x8 to 64x64) and target count.

## Details

- **Category:** non_agentic
- **Style:** CST (constraint satisfaction)
- **Actions:** ACTION6 (click)
- **Tags:** click, spatial_recognition, elimination

## Run

```bash
python games/CST_01/run.py
```
